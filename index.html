About This System

Ground Penetrating Radar (GPR) is widely used in civil engineering, underground utility mapping, archaeology, and safety inspections. However, many commercial GPR systems are expensive, bulky, and power-intensive.

This project introduces a low-cost, portable FMCW GPR prototype operating in the 2.4 GHz ISM band. By using frequency-modulated continuous wave signals, the system achieves stable operation, improved noise resistance, and reliable depth estimation.

Signal processing is performed using software-defined radio (SDR) and Short-Time Fourier Transform (STFT) techniques, producing clear radargra
